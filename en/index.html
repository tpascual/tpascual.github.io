Case studies

Context

Online store launch (Prestashop).

Problem

The store failed in production on launch day.

Solution

  • Fast server-side diagnosis by elimination until the root cause was isolated.
  • Immediate correction without unnecessary structural changes.

Outcome

  • Platform operational in under one hour.
  • Launch unblocked and system stabilized.

Context

Multi-country Shopify e-commerce operations.

Problem

Over-customized theme made every change slow and expensive.

Solution

  • Migration to a standard Shopify 2.0 theme.
  • Core features rebuilt using native capabilities and proven apps.

Outcome

Lower cost and faster delivery of changes, with stable and repeatable operations across countries.

Context

Large-scale commerce project built on API integrations.

Problem

High complexity with a significant risk of misunderstandings and cost overruns.

Solution

  • Extreme decomposition into small, verifiable steps.
  • Continuous validation with real API data and explicit written questions.

Outcome

  • Fact-based decisions.
  • Detected inconsistencies early and justified trade-offs objectively.

Context

Project with minimal and poorly defined requirements.

Problem

Execution based on assumptions without clear validation criteria.

Solution

Proposed a minimum definition baseline and short, testable milestones before scaling execution.

Outcome

  • High effort with low return.
  • Key lesson: without minimum definition, there is no efficient engineering.

Lesson learned: define minimum acceptance criteria before implementation.

About me

I lead e-commerce systems and run production-critical platforms (Shopify Plus + integrations). My focus is operable architecture, clean integrations, and resilient systems that scale without becoming fragile.

Learn more about me

What I do

  • I design, stabilize, and run production e-commerce systems (Shopify Plus + integrations).
  • I work on both new architectures and existing live platforms.
  • I focus on reliability, integration quality, and day-to-day operability.
  • I design end-to-end flows: platform, ERP, logistics, payments, and marketplaces.
  • I document and transfer knowledge so teams can scale safely.

How I create value

  • I design e-commerce architecture for scale, operations, and maintenance from day one.
  • I integrate Shopify Plus and Amazon with ERPs (Navision, M3, SAP, Odoo), logistics, and payments.
  • I reduce manual work, failure points, and operational friction across teams.
  • I lead incident response with postmortems, observability, and standards to improve resilience.

Real context

  • Co-founded Iniciativatec.
  • Former CTO / Technical Director at Julius Commerce Iberica, leading multi-country architecture and integrations.
  • Experience across Europe, USA, and LatAm with multiple brands and stores in parallel.
